Instance Placement Status (API)

Instance Placement Status

An Instance can have three kind of placement status:

  • Instance::PlacementStatus::UNPLACED : the instance doesn't have a meaningful position. It shouldn't be materialized either. It's position therfore shouldn't be used.
  • Instance::PlacementStatus::PLACED : the instance has be placed manually or by an automated tool. It is movable.
  • Instance::PlacementStatus::FIXED : the instance is placed and mustn't be moved by automated tools. It can still be moved manually.
