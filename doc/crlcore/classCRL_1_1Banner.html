Detailed Description

Print Formatted Banners (on ttys).

Display a traditionnal UNIX like banner for tty:

                                              @@     
                             @@@     @@@@@@   @@    
                           @@   @@  @@  @@  @@@@@@@@

                          @@         @@@      @@     
                          @@     @  @@        @@     
                           @@   @@  @@@@@@    @@   @

                                    @     @
